# Xperiq Brand Assets Checklist

**Version 1.0 — June 2026**

Everything that needs to be created before, at, and after launch. Prioritised in order.
Work through this list top-to-bottom. Do not skip ahead.

---

## Priority 1 — Must Have Before Launch

These cannot be skipped. Without these, you cannot launch.

### Logo Files
- [ ] Full logo — horizontal (SVG + PNG @2x)
- [ ] Full logo — stacked (SVG + PNG @2x)
- [ ] Icon only / crystal mark (SVG + PNG @2x)
- [ ] Favicon (16×16, 32×32, 180×180 Apple touch icon — .ico + .png)
- [ ] White version of all above (for dark backgrounds)
- [ ] Black/mono version of all above (for print)

**Who creates this:** Hire a logo designer on Contra, Dribbble, or 99designs. Budget: $300–$1,500.
**Brief to give designer:** See `BRAND_GUIDE.md` — crystal hexagon facet mark + "Xperiq" in Manrope Bold.
**Format to receive:** Vector SVG (editable), PNG at 2x resolution, Figma source file.

### Website / App
- [ ] Open Graph image (1200×630px) — appears when Xperiq links are shared on social
- [ ] Twitter Card image (1200×628px)
- [ ] App icon for browser tab (favicon already listed above)
- [ ] Loading screen / splash logo (for app)

### Social Media Profile Images
Create a profile image for each platform where `@xperiq` handle is registered.

| Platform | Profile image size | Banner/cover size |
|----------|--------------------|-------------------|
| LinkedIn Company Page | 300×300px | 1128×191px |
| X (Twitter) | 400×400px | 1500×500px |
| YouTube | 800×800px | 2560×1440px |
| Instagram | 320×320px | N/A |
| TikTok | 200×200px | N/A |
| GitHub | 420×420px | N/A |

**Profile image:** Crystal mark (icon only) on Xperiq Blue background.
**Cover/banner image:** "Xperiq — Intelligence for Every Experience" wordmark on dark background with subtle crystal facet pattern.

**Tool:** Canva (free) — use 1:1 and banner templates. Upload your logo SVG.

## Asset Storage

Store all assets in one shared folder accessible to the whole team.

**Recommended structure:**
```
Xperiq Brand Assets/
  ├── Logos/
  │   ├── Primary/
  │   ├── White/
  │   └── Mono/
  ├── Colors/
  ├── Fonts/
  ├── Social/
  │   ├── LinkedIn/
  │   ├── X/
  │   ├── YouTube/
  │   └── Instagram/
  ├── Email/
  ├── Presentations/
  ├── Sales/
  └── Screenshots/
```

**Tool:** Google Drive shared folder (free) — share with all team members. Add a `README.txt` at the top level linking to `BRAND_GUIDE.md`.
